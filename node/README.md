### node 启动方式

新项目时候执行一次即可
```shell
npm install express
```

后续项目启动
```shell
npm i
```

启动 localhost:5000/students 服务
```shell
node src/students.js
# 或者
npm run students
```

启动 localhost:5001/cars 服务
```shell
node src/cars.js
# 或者
npm run cars
```

随后浏览器访问验证
